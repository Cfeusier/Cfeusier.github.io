JavaScript (JS) and Client Representation Software

In this guided pairing session, Jay and I got our first taste of working with JS to solve a challenge. We learned quite a bit about the different scenarios that would call for the use of different data structures, particularly arrays and JS objects. We also broke our baby-teeth on JS inheritance via writing methods on a constructor's prototype object. Even though Jay was sicker than a dog and high on cold-medicine, we still managed to finish the challenge. Here was our solution:

function Client(name,age,quote) {
    this.name=name;
    this.age=age;
    this.quote=quote;
}

Client.prototype.display = function() {
  return "name: " + this.name + "\n" + "age: " + this.age + "\n" + "quote: " + this.quote;
};

Client.prototype.showQuote = function() {
  return this.quote;
};

var clark = new Client("Clark Feusier", 26, "I don't think you're ready for this jelly.");
